# 📌 Markdown 常用符号清单  

## ✅ 状态 / 任务清单
| 符号 | 含义 |
|------|------|
| ✅ | 已完成 |
| ☑️ | 已勾选 |
| ✔️ | 完成/确认 |
| ❌ | 未完成/错误 |
| ✖️ | 关闭/拒绝 |
| ⭕ | 待确认 |
| 🔲 | 待处理 |

---

## 🚀 提示 / 强调
| 符号 | 含义 |
|------|------|
| 🚀 | 重点/推荐 |
| ⚡ | 高效/加速 |
| 🔥 | 热点/重要 |
| 💡 | 灵感/提示 |
| 📌 | 固定/重点标记 |
| 📍 | 位置/定位 |
| ⭐ | 星标/重点 |
| 🎯 | 目标 |

---

## ⚠️ 警告 / 注意
| 符号 | 含义 |
|------|------|
| ⚠️ | 注意事项 |
| ❗ | 强调警告 |
| ❕ | 次要提醒 |
| 🔒 | 安全/加密 |
| 🛑 | 停止/禁止 |
| ⛔ | 禁用 |
| 🟡 | 警告 (黄色标记) |
| 🔴 | 严重问题 (红色标记) |
